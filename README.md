# 计算共形几何 (Computational Conformal Geometry) 学习笔记

本仓库包含顾险峰老师「计算共形几何系列课程」的学习笔记，课程来源为 [Bilibili BV1UC4y187D4](https://www.bilibili.com/video/BV1UC4y187D4/)。

## 课程简介

本系列课程介绍了代数拓扑、曲面微分几何、黎曼面和几何偏微分方程等领域的基本概念和定理，讲解了曲面同伦群、同调群、调和映射、亚纯微分、叶状结构、共形映射、拟共形映射和曲率流的计算方法，并简介这些理论在计算机图形学、计算机视觉、可视化、几何建模、无线传感器网络、医学图像和深度学习中的应用。
